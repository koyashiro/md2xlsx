# md2xlsx

## Installation

```sh
go install github.com/koyashiro/md2xlsx
```

## Usage

```sh
md2xlsx INPUT
```

## Example

example spec file is [here.md](https://github.com/koyashiro/md2xlsx/blob/develop/example/spec.md)

```sh
md2xlsx spec.md
```
